ChooseBaseDirOfFile (name[, pref_base=system.ResourceDirectory])
Chooses a Corona base directory for a file. If the file is found under the "preferred" directory, that one is used; otherwise, system.ResourceDirectory is used (n.b. the file may or may not exist, in this case).

This is intended for simulator-side tests. On a device, system.ResourceDirectory is always chosen.

Parameters:

  • name string File path.
  • pref_base Preferred directory base. (default system.ResourceDirectory)

Returns:

    userdata If chosen, pref_base; otherwise, system.ResourceDirectory.

EnumerateFiles (path[, opts], into)
Enumerates files in a given directory.

Parameters:

  • path string Directory path.
  • opts table

    Enumeration options. Fields:

    • exts: Extensions filter. If this is a string, only files ending in the string are enumerated. If it is an array, only files ending in one of its strings (tried in order) are enumerated. Otherwise, all files are enumerated.
    • base: Directory base. If absent, system.ResourcesDirectory.
  • into table If provided, files are appended here. Otherwise, a table is provided.

Returns:

    table Enumerated files.

PutInTrash_Guard (name, object[, base=system.ResourceDirectory])
Variant of PutInTrash which accepts a display object as a "guard".

The guard is assumed to depend upon the file, e.g. the latter supplied its assets. Thus, until the guard is removed, the trash logic will leave the file alone. The file may cease to exist via external means, of course, in which case its state is evicted.

Parameters:

  • name string
  • object DisplayObject
  • base (default system.ResourceDirectory)
WatchForFileModification (path, func[, opts])
Launches a timer to watch a file or directory for modifications.

Parameters:

  • path string File or directory path.
  • func callable

    On modification, this is called as func(path, how), where how is one of:

    • "created": File was created (or re-created) once watching was begun.
    • "deleted": File was deleted once watching was begun.
    • "modified": File was modified while being watched.
  • opts table

    Watch options. Fields:

    • exts: As per EnumerateFiles.
    • base: As per EnumerateFiles.
    • get_contents: DDDD

Returns:

    TimerHandle A timer, which may be cancelled.
