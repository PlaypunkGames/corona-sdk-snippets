Module spline_ops.cubic

Various cubic splines and related functionality.

In several of the routines, the spline type will be one of "bezier", "catmull_rom", or "hermite".

For purposes of this module, an instance of type Coeffs is a value, e.g. a table, that has and / or receives number members a, b, c, d, whereas for a Vector the members are x and y.

BezierToHermite (src1, src2, src3, src4[, dst1=src1[, dst2=src2[, dst3=src3[, dst4=src4]]]])
Converts coefficients from Bézier (P1, Q1, Q2, P2) to Hermite (P1, P2, T1, T2) form.

This (and the similar functions in this module) are written so that output coefficients may safely overwrite the inputs, i.e. if src* = dst*.

Parameters:

  • src1 Vector Vector #1 (i.e. P1)...
  • src2 Vector ...#2 (Q1)...
  • src3 Vector ...#3 (Q2)...
  • src4 Vector ...and #4 (P2).
  • dst1 Vector Target vector #1 (i.e. will receive P1)... (default src1)
  • dst2 Vector ...#2 (P2)... (default src2)
  • dst3 Vector ...#3 (T1)... (default src3)
  • dst4 Vector ...and #4 (T2). (default src4)

GetPolyCoeffs (stype, a, b, c, d)
Gets polynomial coefficients for calculating line integrands.

A given cubic spline can be written as a polynomial Ax³ + Bx² + Cx + D, for both its x- and y-components. Furthermore, its derivative is useful for computing the arc length of the spline.

Parameters:

  • stype string Spline type.
  • a Vector Vector #1 defining the spline...
  • b Vector ...#2...
  • c Vector ...#3...
  • d Vector ...and #4.

Returns:

  1. number A, for x...
  2. number ...and y.
  3. number B, for x...
  4. number ...and y.
  5. number C, for x...
  6. number ...and y.

See also:

  • LineIntegrand
  • SetPolyFromCoeffs

LineIntegrand ([poly])
Line integrand for a cubic polynomial.

Parameters:

  • poly array The underlying polynomial, (dx/dt)² + (dy/dt)²: elements 1 to 5 are the x⁴, x³, x², x, and constant coefficients, respectively. If absent, a table is supplied.

Returns:

  1. function Integrand function, which may be passed e.g. as the func argument to the various integrators.
  2. array poly.

See also:

    SetPolyFromCoeffs

SetPolyFromCoeffs (poly, ax, ay, bx, by, cx, cy)
Assigns integrand coefficents (in particular, as expected by LineIntegrand's integrand function), given a cubic polynomial's derivatives: dx/dt = 3Ax² + 2Bx + C, dy/dt = 3Dy² + 2Ey + F.

Parameters:

  • poly array Polynomial.

Truncate (src1, src2, src3, src4, t1, t2[, dst1=src1[, dst2=src2[, dst3=src3[, dst4=src4]]]])
Truncates a Bézier spline, i.e. the part of the spline ∈ [t1, t2] becomes a new Bézier spline, reparameterized to the interval [0, 1].

Parameters:

  • src1 Vector Vector #1 (i.e. P1)...
  • src2 Vector ...#2 (Q1)...
  • src3 Vector ...#3 (Q2)...
  • src4 Vector ...and #4 (P2).
  • t1 number Lower bound of new interval, ∈ [0, t2).
  • t2 number Upper bound of new interval, ∈ (t1, 1].
  • dst1 Vector Target vector #1 (i.e. will receive P1)... (default src1)
  • dst2 Vector ...#2 (Q1)... (default src2)
  • dst3 Vector ...#3 (Q2)... (default src3)
  • dst4 Vector ...and #4 (P2). (default src4)
